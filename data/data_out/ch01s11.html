Полезные ресурсы

В Интернете полно полезных ресурсов для разработчиков Python. Наиболее важные и очевидные из них уже упоминались ранее, но здесь они повторяются, чтобы этот список был последовательным:

  • Документация Python
  • PyPI - указатель пакета Python
  • PEP 0 - Указатель предложений по усовершенствованию Python

Другие ресурсы, такие как книги и учебные пособия, полезны, но часто очень быстро устаревают. Что не устаревает, так это ресурсы, которые активно курируются сообществом или периодически выпускаются. Два, которые стоит порекомендовать, это:

  • Awesome-python ( https://github.com/vinta/awesome-python ), который включает тщательно подобранный список популярных пакетов и фреймворков.
  • Python Weekly ( http://www.pythonweekly.com/ ) - популярный информационный бюллетень, который каждую неделю доставляет своим подписчикам десятки новых и интересных пакетов и ресурсов Python.

Эти два ресурса предоставят читателю массу дополнительных материалов для чтения в течение нескольких месяцев.
